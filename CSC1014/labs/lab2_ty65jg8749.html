Week 2 Lab – Types, Expressions & the Python Calculator

In this lab you will practise using Python as a calculator, explore how different types behave, and get more comfortable reading and writing expressions with a clear order of operations.

Lab goals
  • Use type() to inspect the types of values and expression results.
  • Practise writing and evaluating arithmetic expressions.
  • Develop intuition for operator precedence and the effect of parentheses.
  • Use Python to solve small calculator-style word problems.
  • Write and interpret simple comparison expressions that yield True/False.
  • Clearly distinguish between numbers and strings.
Important Today we still avoid variables and input. All your work will be with literal values inside expressions. Next week, you’ll start giving those values names.

          Error preview
          In Lab 1 you saw SyntaxError when the structure of the code
          was broken.  
          In this exercise you probably saw a TypeError, which happens when
          you try an operation that doesn’t make sense for the types involved
          (like subtracting one string from another).
          We’ll talk more about different error types later in the course.
