Lab 3 – Variables & State

In this lab you will practise using variables as “named boxes”, understand assignment and re-assignment, trace how values change, and use simple counter and accumulator patterns.

Lab goals
  • Define and update variables to model simple situations.
  • Trace variable values step by step through a small script.
  • Use the counter and accumulator patterns (without loops yet).
  • Choose good variable names and recognise illegal ones.
  • Understand common mistakes like using a variable before assignment.
