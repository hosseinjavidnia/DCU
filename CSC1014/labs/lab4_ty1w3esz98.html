Lab 4 – Input, Output & Simple Interactive Programs

In this lab you will practise talking to the user using print() and input(), converting text input to numbers, and building small interactive Python scripts.

Lab goals
  • Write your own print() statements (including basic formatting).
  • Use input() to read text from the user.
  • Convert text to numbers with int() and float().
  • Combine these to create small interactive programs.
  • See how errors like TypeError and ValueError arise with bad types or inputs.

Exercise 0 – Warm-up: Printing Variants

Practise writing different forms of print(). For each sub-task, write the full print() line yourself, then run and check the output.
