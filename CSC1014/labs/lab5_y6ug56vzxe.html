Lab 5 – Making Decisions with if Statements

In this lab you will practise writing Boolean expressions and using if, if-else, and if-elif-else to make decisions in your programs.

Lab goals
  • Write and test Boolean expressions using comparison and logical operators.
  • Use one-way, two-way, and multi-way conditionals.
  • Combine simple conditions with and, or, and not.
  • Debug common conditional mistakes (missing colons, = vs ==, etc.).
  • Build small decision-based scripts from informal descriptions.
Allowed tools in this lab
  • Variables, expressions, and types from earlier weeks.
  • Input/output: input(), print(), f-strings, int(), float().
  • Comparisons: ==, !=, <, <=, >, >=.
  • Logical operators: and, or, not.
  • Conditionals: if, if-else, if-elif-else.
