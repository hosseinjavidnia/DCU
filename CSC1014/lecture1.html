The first thing any programmer learns is how to make the computer speak. In Python, we use the print() function. Think of functions as "verbs"—they are actions the computer performs.

3. Numbers vs Strings

        

          Computers are excellent calculators. When working with numbers, we do not use quotes.
          If you put quotes around a number (e.g., "5"), Python treats it as a text character.
        


        

          Key Concept
          Integers are whole numbers. Strings are text. You can add numbers together, but adding strings just joins them together (concatenation).
