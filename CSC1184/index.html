Lecture Notes

Lecture 1

Introduction & Data Analysis Mindset

In Lecture 1, we set the scene: what “programming for data analysis” means, how a typical analysis workflow looks, and warm up with small Python examples using lists and built-in functions.

Lecture 2

Files, CSV & List-of-Dicts

This week we move from hard-coded lists to data in files. We look at how CSVs are structured, how to read them safely with with open(...), and how to turn rows into labelled dictionaries using zip. We finish by sketching a small reusable load_data(...) pattern.

Lecture 3

NumPy Arrays & Vectorised Operations

We move from plain Python lists to NumPy arrays: creating arrays, understanding shape and dtype, using masks, and applying fast vectorised operations for summaries and simple simulations.

pandas DataFrames & Table Operations

Lab Assignments

Lab 1

Lists, Built-ins & Simple Loops

Hands-on practice treating Python lists as tiny datasets using built-ins (len, sum, min, max) and simple loops for counting, cleaning, simple transforms, and basic smoothing.

Locked
Lab 2

Dictionaries & File Loading

Practice building dictionaries from lists, then write a small load_data(...) function that reads a CSV file, uses zip to create row dictionaries, and handles simple “messy data” cases without crashing.

Locked
Lab 3

NumPy Practice: Masks & Simulation

Hands-on practice with NumPy arrays: converting from lists and dicts, using boolean masks, comparing groups, and running a tiny simulation of exam scores.

Locked
Lab 4

pandas: Filtering & Summaries

Hands-on practice with pandas: build a small student table, explore it with head/info/describe, select rows and columns, filter by conditions, create new columns, and compute simple grouped averages.
