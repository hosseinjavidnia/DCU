Goals & Suggested Schedule

Learning goals

  • Use lists as tiny datasets (one list = one meaning).
  • Apply built-ins: len, sum, min, max, print.
  • Write for loops for counting, cleaning, and simple transforms.
How the TODOs work Placeholders are written as __TODO__. Replace every __TODO__ with real code before you click Run. Leaving them as None will produce a clear error (TypeError), which is expected until you complete the step.
